>faq Lang
I was made in TypeScript, C++ and Rust. MochEngine1 is written in Rust and MochEngine2 is primarily written in C++ with bindings to Rust for the TETR.IO compatible engine library. TypeScript is used for the main client.

>faq Gravity
Gravity is currently officially unsupported, and causes misdrops and desync issues, especially in this release. It CAN be enabled in a room using administrative overrides, but it is not recommended.

>faq Workers
A worker is an equivalent to a thread of MochBot, and helps with load balancing, especially under high use.

Workers are setup on nodes, which are entire machines/servers dedicated to MochBot!

For example, you are currently on proton running in High performance mode.
